Catatan Untuk Program 

Dihalaman MENU UTAMA terdapat 4 opsi : 
Opsi 1 : Untuk pergi ke halaman admin
Opsi 2 : Untuk pergi ke halaman mahasiswa
Opsi 3 : Untuk melihat nama dan nilai peserta yang sudah mengerjakan quis
Opsi DONE : Untuk mengakhiri program

Dihalaman ADMIN terdapat 5 opsi :
Opsi 1 : Menambahkan Pertanyaan di Bank Soal
Opsi 2 : Mengganti Kunci jawaban / mengedit Soal
Opsi 3 : Menghapus Soal dari Bank Soal
Opsi 4 : Untuk kembali ke halamam MENU UTAMA
Opsi 5 : Untuk mengetahui jenis soal Mudah/Sulit menggunakan sorting yang sudah dikerjakan oleh peserta

Diopsi Add Question berisi :
1. Panduan Mengisi Soal 
2. Menginput Soal dan Kunci Jawaban 
3. Mengakhiri dengan input STOP 
Dan kembali ke halaman MENU UTAMA

Diopsi Change or Edit Question berisi : 
Kita dapat mengedit soal dan kunci jawaban 

Diopsi Delete Question berisi :
Menghapus soal yang sudah ada dan sudah ditambahkan dengan minimal 5 soal yang terdapat di banksoal

Diopsi Back Home menu untuk kembali ke halaman MENU UTAMA

Diopsi Info mengenai soal untuk melihat soal yang ada didalam Banksoal


Dihalaman MAHASISWA terdapat Opsi :
1. Halaman Registrasi
2. Halaman Login
3. Halaman Kembali

Diopsi Registrasi : 
Peserta mengisi Username dan Password yang harus diingat untuk diinput dihalaman Login

Diopsi Login :
Peseta kembali mengisi usernam dan password yang sudah dibuat di halaman registrasi
Setalah username dan password sama peserta akan ditanyakan akan mengikuti kuis atau tidak
Setelah menjawab ingin mengikuti kuis peserta akan langsung menuju halaman kuis
Dan Pesrta diperbolehkan mengulang pengerjaan hingga puas

Dihalaman Kuis :
Peserta akan menjawab pertanyaan yang sudah ada dan berakhir ketika sudah menjawab semua pertanyaan 
Juga Nilai peserta akan muncul diakhir kuis

Diopsi Back to home menu :
Peserta akan kembali ke halaman MENU UTAMA


Dihalaman Melihat nama dan nilai peserta berisi :
nama username dan nilai peserta yang sudah menjawab pertanyaan dan nilai dari pengerjaan akhir peserta 
nilai yang diambil adalah nilai terakhir pengerjaan (Last Attempt)

Diopsi DONE : Untuk mengakhiri Proram KUIS
